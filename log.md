## Installation

```sh
pip install mkdocs-material
```

And init project:

```sh
mkdocs new .
```

### Commands

- Serve in LAN:
<https://www.mkdocs.org/user-guide/cli/#mkdocs-serve>

```sh
mkdocs serve -a 192.168.1.91:8181
```

- Serve cold from project directory to LAN:

```sh
source ./activate && mkdocs serve -a 192.168.1.91:8181
```
